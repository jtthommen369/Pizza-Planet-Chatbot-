# Pizza Planet Chatbot

A Python-based interactive chatbot that simulates a pizza ordering experience. Demonstrates string matching, conditional logic, and basic natural language handling using Python.

## How to Run
1. Clone the repository:

git clone https://github.com/jtthommen369/Pizza-Planet-Chatbot.git

2. Navigate to the folder and run:

python chatbot.py

3. Interact with the chatbot in the terminal.

## Features
- Conditional logic for different pizza sizes and toppings
- Recognizes zip codes to finalize order
- Handles unexpected input with fallback response
